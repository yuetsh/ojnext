## 第一： print 后面跟着英文括号，不要写等号！！！

### 下面是错误的：

```py
print=("China")
```

## 第二： 所有的符号都是英文的

### 下面是错误的：

```py
print（"China"）
print（“China”）
```


## 第三：下面的写法是等价的

### 第一种：
```py
a="China"
print(a)
```

### 第二种：
```py
print("China")
```

## 第四：输出有几行就需要几个print()

### 输出三行你好：

```py
print("你好")
print("你好")
print("你好")
```

## 第五：每行输出多个

```py
print("陈旭", "赵铖阳")
```
