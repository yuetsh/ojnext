遇到以下情况：

## 输入的文字，用空格隔开。比如：你好 世界

你输入的`"你好 世界"`是一句话，但是你需要得到 `"你好"` 和 `"世界"`，并分别赋值给两个变量a和b。

你需要这样写：

```py
a,b=input().split()
```

如果是三个用空格隔开，比如：小明 小华 小王

```py
a,b,c=input().split()
```

那 `a` 代表 小明，`b` 代表 小华，`c` 代表 小王

## 输入两个整数，用空格隔开。比如：10 20

写法如下：

```py
n=input().split()

a=int(n[0])
b=int(n[1])
```

如果是三个整数，以此类推：

```py
n=input().split()

a=int(n[0])
b=int(n[1])
c=int(n[2])
```

## 要是小数。比如：2.33 5.18

写法如下：

```py
n=input().split()

a=float(n[0])
b=float(n[1])
```
